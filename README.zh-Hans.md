### Summary
Strict DLP Chinese (SDC) 是基于官方 eMule Xtreme DLP 的严格版本。在官方 eMule Xtreme DLP 的基础上，将 VeryCD-Mod 和 VeryCD-EasyMule-Mod (第1版) 中的部分或全部加入了 SoftBan 列表，原因是 GPL-breaker、私有网络和社区客户端等不良行为。
